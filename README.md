# 智慧郵件分類系統

基於機器學習的垃圾郵件分類系統，使用 Streamlit 建立互動式介面。

## 功能特色

- 📊 即時垃圾郵件檢測
- 📈 詳細分析報告
- 🔍 文本特徵分析
- 💡 預測可信度評估

## 安裝與執行

1. 安裝相依套件：
```bash
pip install -r requirements.txt
```

2. 執行應用程式：
```bash
streamlit run app.py
```

## 系統需求

- Python 3.8+
- 相關套件參見 requirements.txt

## 使用方法

1. 在文字框中輸入要檢查的郵件內容
2. 點擊「分析」按鈕
3. 系統會顯示預測結果和機率分布

## 資料來源

本專案使用 SMS Spam Collection Dataset
